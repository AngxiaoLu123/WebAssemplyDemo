# WebAssemplyDemo

## Compile

Install .Net 6. Open command prompt, cd to WebAssemplyDemo\WebAssemblyTest run 

`
dotnet publish
`

Copy dotnet.d.ts and dotnet.js from WebAssemplyDemo\WebAssemblyTest\WebAssemblyTest\bin to WebAssemplyDemo\src\assets
